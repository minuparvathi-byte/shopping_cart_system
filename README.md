# shopping_cart_system
# Python Shopping Cart List Operations

## Overview
This project demonstrates basic Python list operations using a shopping cart example. It helps beginners understand how to manipulate lists by adding, removing, and accessing elements.

## Features
- Add multiple items using `extend()`
- Insert an item at a specific position using `insert()`
- Remove an item using `remove()`
- Remove the last item using `pop()`
- Find the index of an item using `index()`
- Count occurrences of an item using `count()`
- Access items using indexing
- Retrieve items using slicing
- Find the total number of items using `len()`

Python Lists
List Methods

## Code Example

```python
Shopping_cart = ["Milk", "Bread", "Eggs", "Apple"]

# Extend list
Shopping_cart.extend(["Oil", "Jam"])

# Insert item
Shopping_cart.insert(2, "Batter")

# Remove item
Shopping_cart.remove("Apple")

# Pop last item
Shopping_cart.pop()
